# Spring-FrameworkCodes

Spring have Three Moduls: IoC / AOP / MVC

IoC - Inversion of Controller.
  - it is a container that creates beans and maintain there Lifecycles.
    
Spring IoC have a two ways to create beans:
1. BeanFactory (Interface) - supports XML based Config.
2. ApplicationContext (Interface) - supports both XML and Annotation Based

--> BeanFactory replaced by ApplicationContext in Spring v-4.
